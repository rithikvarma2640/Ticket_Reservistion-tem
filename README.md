#🎟️ Ticket Reservation System (C++)
📌 Overview

This project is a console-based Ticket Reservation System developed using C++ and the Queue (FIFO) data structure.
It simulates real-world ticket booking operations by managing customer requests in an organized and efficient manner.

🚀 Features
🎫 Ticket Booking (Enqueue operation)
👤 Customer Servicing (Dequeue operation)
📋 Display Waiting List
⚠️ Overflow & Underflow Handling
🖥️ Menu-Driven User Interface
⏱️ Real-Time Processing of Requests
🛠️ Tech Stack
Language: C++
Concepts Used:
Queue (FIFO)
Data Structures & Algorithms
Conditional Statements
Menu-Driven Programming
⚙️ How It Works
Users can book tickets, which are added to the queue.
Customers are served in First-In-First-Out (FIFO) order.
The system maintains and displays a waiting list.
Proper checks are implemented to handle:
Queue Overflow (Full capacity)
Queue Underflow (Empty queue)
▶️ How to Run
g++ ticket.cpp -o ticket
./ticket
📷 Sample Menu
1. Book Ticket
2. Serve Customer
3. Display Waiting List
4. Exit
